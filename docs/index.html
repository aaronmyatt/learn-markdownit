basic usage

Straight from the repo. Import the library, instantiate, render some markdown!

import markdownit from "npm:markdown-it";

input.mdi = markdownit()
$p.set(input, '/markdown/basics', input.mdi.render('# markdown-it rulezz!'))

render inline

Under what circumstances would one want to render markdown without line breaks? To render the inline syntax like italics and bold? Maybe offer your user only a subset of the syntax in a text box/wysiwyg style editor?

$p.set(input, '/markdown/basicInline', input.mdi.renderInline('# markdown-it rulezz! __mdi__ *mdi*'))

The markdown-it object is noisy as hell, tons of unicode and regex blobs. Seems like most of it is related to matching urls/links/emails. Let's just output the top level keys.

input.mdi = Object.keys(input.mdi);
