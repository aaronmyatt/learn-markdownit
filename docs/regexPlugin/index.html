Regex Plugins for markdown-it

So there's a 9 year old markdown-it plugin that helps to make markdown-it plugins easier. Unfortunately, it doesn't seem to be maintained and likely isn't playing nicely with ESM/Deno/Typescript. So I'm going to try and modernise it.

In principle it is quite simple, pass it a regex, and it'll pass you the markdown-it parser tokens that match. The challenge.. how to make it with Pipedown!

import markdownit from "npm:markdown-it";

    "build": ["esm"]
